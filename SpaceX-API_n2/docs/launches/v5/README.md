## Changes from v4 -> v5

* Crew is now an array of objects, to allow for more data on an individual launch for a crew member

### Old Format

```json
{
  "crew": [
    "1234567890",
    "123456789",
    "123456789",
  ]
}
```

### New Format

```json
{
  "crew": [
    "1234567890",
    "123456789",
    "123456789",
  ]
}
```
