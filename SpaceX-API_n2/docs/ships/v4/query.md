# Query ships

**Method** : `POST`

**URL** : `https://api.spacexdata.com/v4/ships/query`

**Auth required** : `False`

**Body** :

See [query](../../queries.md) guide for more details on building queries and paginating results.

```json
{
  "query": {},
  "options": {}
}
```
