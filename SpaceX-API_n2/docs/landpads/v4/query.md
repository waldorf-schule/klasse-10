# Query landing pads

**Method** : `POST`

**URL** : `https://api.spacexdata.com/v4/landpads/query`

**Auth required** : `False`

**Body** :

See [query](../../queries.md) guide for more details on building queries and paginating results.

```json
{
  "query": {},
  "options": {}
}
```
